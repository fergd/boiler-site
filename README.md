Boiler Site
===========

A very barbones startup environment. Literally no frills. Good for trying out an idea super quick like. 

What's in the box?
----

  - Basic HTML file with boilerplate markup
  - Gulp task runner that contains
    - local server
    - live reload

Version
----

1.0

Installation
