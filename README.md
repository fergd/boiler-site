## Use

### Watching and viewing on localhost
-In your terminal, run ` $ grunt ` in your project directory: 
-Navigate to `localhost:8000` to view your project.

### Deploying to gh-pages
- Run ` grunt deploy `
- View your app on your custom gh-pages URL
